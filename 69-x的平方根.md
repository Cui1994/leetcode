# x的平方根

## 题目

>实现 int sqrt(int x) 函数。
>计算并返回 x 的平方根，其中 x 是非负整数。
>由于返回类型是整数，结果只保留整数的部分，小数部分将被舍去。

## 思路

采用二分查找，当mid的平方小于x，mid+1的平方大于x即为平方根的整数值，要注意整形溢出。

## Code
```java
class Solution {
    public int mySqrt(int x) {
        int low = 0;
        int high = x;
        int mid;

        if (x == 1) return x;

        while (low <= high) {

            mid = low + (high - low) / 2;

            if ((long)mid*mid == x) {
                return mid;
            }
            else if ((long)mid*mid < x) {
                if ((long)(mid+1)*(mid+1) > x) return mid;
                else low = mid + 1;
            }
            else {
                high = mid - 1;
            }
        }

        return -1;
    }
}
```

## 复杂度分析

O(logN)
